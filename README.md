# Путешествия по России

Проектная работа, выполненная при обучении
на [Яндекс.Практикуме](https://praktikum.yandex.ru/)
по специальности Веб-разработчик.

**Тема: HTML и CSS — адаптивная вёрстка и работа с макетом**
